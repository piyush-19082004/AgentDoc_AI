# AgentDoc Frontend

Next.js frontend for AgentDoc AI. Uses TailwindCSS, Framer Motion, and Axios to interact with the FastAPI backend.

Quick start:

1. cd frontend
2. npm install
3. Set environment variable `NEXT_PUBLIC_API_URL` to your backend base url (e.g. `https://agent-doc-...vercel.app`)
4. npm run dev
