# AgentDoc AI

> An Autonomous Multi-Agent AI System that converts natural language requests into professional Microsoft Word documents using FastAPI, Groq LLM, and python-docx.

---

## Overview

AgentDoc AI is a modular AI agent designed to automate document creation from natural language requests.

Instead of generating a document in a single LLM call, the system follows an autonomous multi-agent workflow:

- Plan the work
- Execute tasks sequentially
- Review the generated content
- Generate a professionally formatted Word document

This architecture improves modularity, maintainability, and extensibility while demonstrating autonomous planning and reasoning.

---

## Features

- Autonomous task planning

# Agent Workflow

## 1. Planner

The Planner Agent receives the user's request and generates a structured execution plan.

Example:

```
User Request
      │
      ▼
Execution Plan

Task 1
Task 2
Task 3
Task 4
```

---

## 2. Executor

The Executor executes each task in dependency order.

Each completed task becomes context for future tasks.

```
Task 1
   │
Task 2
   │
Task 3
   │
Task 4
```

---

## 3. Reflection

The Reflection Agent evaluates the generated content for:

- Missing sections
- Quality improvements
- Consistency
- Completeness

---

## 4. Document Generator

Finally, all outputs are converted into a professionally formatted Microsoft Word document.

Generated document includes:

- Executive Summary
- Assumptions
- Task Outputs
- Recommendations
- Timestamp

# Engineering Decisions

### Why Multi-Agent?

Instead of asking the LLM to generate the complete proposal in one prompt, the project separates the workflow into four independent agents.

Benefits:

- Better modularity
- Easier debugging
- Easier future extension
- Clear execution flow

---

### Why Sequential Execution?

Sequential execution allows each task to use outputs generated by previous tasks.

This makes reasoning more reliable than independent parallel execution.

---

### Why FastAPI?

- High performance
- Automatic Swagger documentation
- Pydantic validation
- Easy deployment
